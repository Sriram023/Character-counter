# Character Counter Web App

## 📌 Project Overview

This is a simple and interactive **Character Counter Web Application**
built using HTML, CSS, and JavaScript.\
It allows users to type a message inside a textarea and dynamically
displays the number of characters typed out of a maximum limit (200
characters).

When the character limit is reached, a warning message is displayed.

------------------------------------------------------------------------

## 🚀 Features

-   Live character counting
-   Maximum limit of 200 characters
-   Warning message when limit is reached
-   Clean and simple UI
-   Beginner-friendly JavaScript logic
-   Responsive basic layout

------------------------------------------------------------------------

## 🛠️ Technologies Used

-   HTML5
-   CSS3
-   JavaScript (Vanilla JS)

## 💡 How It Works

1.  The textarea has a `maxlength` attribute set to 200.
2.  An event listener (`input`) tracks every keystroke.
3.  The script calculates the current length of the text.
4.  The counter updates dynamically.
5.  When the limit is reached, a warning message appears.

------------------------------------------------------------------------

## ▶️ How to Run

1.  Download or clone the project.
2.  Open the `index.html` file in your browser.
3.  Start typing inside the message box.
4.  Watch the live character counter update.

------------------------------------------------------------------------

## 📸 Output Behavior

-   Displays: `0 / 200 characters` initially.
-   Updates count as user types.
-   Shows red warning message when 200 characters are reached.
